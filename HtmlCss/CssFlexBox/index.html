FLEX PROPERTIES

  • display:flex | inline-felx :- whichi allow to display the container as flex
  • flex-direction : row | column :- We can assign div to vertically or horizontally.
  • flax-wrap: wrap :nowrap | wrapreverse :-
  • flex-basis: it's a simple width for flex(container)
  • justify-content:flex-start | flex-end | center :- Alignment for flex box
  • align-self: flex-start|flex-end|center :- align-self hide the over data in the div
  • align-items:flex-start|flex-end|center :- to manage the items inside the flex
  • align-content: flex-start|flex-end|center : to mange the content
